Not sure what to write about for your next blog post? Maybe this Dashboard Widget can inspire you.

== Description ==

In Chris Lema's blog post "The simple truth that will help you write blog posts faster" (http://chrislema.com/write-blog-posts-faster/), Chris talks about the 7 main blog post topics used. This widget simply puts them in your dashboard to offer you a writing prompt when logging in.

== Installation ==

1. Install the plugin
2. Activate the plugin
3. Visit your Dashboard
4. Get Inspired!
5. WRITE!
